Musify
Client Side Web Programming
JavaScript, React, HTML, CSS
This is a mimic web site of Spotify, parsing Spotify API. With jQuery library, React, and boostrap, and a public Web API, users can interactively search music from this website.<br />This website also allows users to listen to preview music by clicking the results, search for albums and playlists related to the search keyword, and provide links to Spotify for users to enjoy music.
